Git and Github Resources

To follow along with my in-class demo on version control, you'll need to do the following:
  1. Sign up for a GitHub Account.
  2. Download the GitHub GUI for Mac or Windows. (If you use Linux, you can install git using your package manager and either use the terminal or a different GUI such as git-cola.)
I'll give a basic introduction to git and GitHub in class. If you want to learn more, I suggest taking a look at the following references:
  • Try Git - An interactive tutorial covering git command line basics.
  • Pro Git - A free and comprehensive guide to git.
  • GitHub Guides
  • Pull Request Tutorial

Rcpp and RcppArmadillo

In class, I'll give you a quick introduction to writing high-performance C++ code and running it from R using the fantastic Rcpp and RcppArmadillo packages. I've posted some instructions for how to get you machine setup to use these packages on the website for my
  • Warren Center R Workshop. If you have any trouble, let me know and I'll gelp you get things up and running. If you've never done any C or C++ programming before, check out Hadley Wickham's High Performance Functions with Rcpp. It's a quick and dirty guide to the minimum you'll need to know to get started with Rcpp. The RcppArmadillo package gives you access to the fantastic Armadillo C++ Library, a high-performance, open-source linear algebra library with syntax that is "deliberately similar to Matlab." Full documentation for the Armadillo library is available here. If you'd like to see some more complicated and interesting examples of Rcpp and RcppArmadillo in practice, check out the Rcpp Gallery.
